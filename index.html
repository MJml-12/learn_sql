Database liga fiktif langsung jalan di browser (SQLite) · Ketik query di kotak bawah, tekan Jalankan (atau Ctrl+Enter)
Skema database (liga fiktif 8 klub, musim 2025):
teams(team_id, name, city, stadium_capacity)
players(player_id, team_id, name, position, birth_year, nationality) — posisi: GK/DF/MF/FW
matches(match_id, match_date, home_team_id, away_team_id, home_goals, away_goals, attendance)
goals(goal_id, match_id, player_id, minute, is_penalty)
Coba dulu: SELECT * FROM teams;

MINGGU 1 — Fondasi

Hari 1 · SELECT, WHERE, ORDER BY
1-1. Tampilkan semua pemain posisi FW, urutkan dari yang termuda.
Solusi
SELECT name, birth_year, nationality
FROM players
WHERE position = 'FW'
ORDER BY birth_year DESC;
1-2. Pertandingan dengan penonton di atas 30.000, tampilkan tanggal dan attendance saja.
Solusi
SELECT match_date, attendance
FROM matches
WHERE attendance > 30000
ORDER BY attendance DESC;
1-3. Pemain non-Jepang (nationality bukan 'Japan'), 10 pertama berdasarkan nama.
Solusi
SELECT name, nationality
FROM players
WHERE nationality <> 'Japan'
ORDER BY name
LIMIT 10;
1-4. Pemain MF atau FW yang lahir tahun 2000 ke atas (kombinasi AND/OR — hati-hati pakai kurung).
Solusi
SELECT name, position, birth_year
FROM players
WHERE (position = 'MF' OR position = 'FW')
  AND birth_year >= 2000
ORDER BY birth_year DESC;
1-5. Pertandingan bulan Mei 2025 saja (petunjuk: LIKE '2025-05%' atau BETWEEN).
Solusi
SELECT match_date, home_team_id, away_team_id, home_goals, away_goals
FROM matches
WHERE match_date LIKE '2025-05%'
ORDER BY match_date;
1-6. Pemain dari Brazil, South Korea, atau Indonesia (pakai IN).
Solusi
SELECT name, position, nationality
FROM players
WHERE nationality IN ('Brazil', 'South Korea', 'Indonesia')
ORDER BY nationality, name;
Hari 2 · Agregasi: COUNT, SUM, AVG, GROUP BY, HAVING
2-1. Berapa jumlah pemain per posisi?
Solusi
SELECT position, COUNT(*) AS jumlah
FROM players
GROUP BY position;
2-2. Rata-rata penonton per pertandingan (bulatkan).
Solusi
SELECT ROUND(AVG(attendance)) AS rata2_penonton
FROM matches;
2-3. Kewarganegaraan yang punya lebih dari 3 pemain di liga.
Solusi
SELECT nationality, COUNT(*) AS n
FROM players
GROUP BY nationality
HAVING COUNT(*) > 3
ORDER BY n DESC;
2-4. Total gol per bulan (petunjuk: strftime('%m', match_date) lewat join ke matches — boleh tunda ke Hari 3 kalau belum bisa join).
Solusi
SELECT strftime('%Y-%m', m.match_date) AS bulan,
       COUNT(g.goal_id) AS total_gol
FROM goals g
JOIN matches m ON m.match_id = g.match_id
GROUP BY bulan
ORDER BY bulan;
2-5. Total gol kandang dan total gol tandang seluruh liga (SUM dua kolom sekaligus).
Solusi
SELECT SUM(home_goals) AS gol_kandang,
       SUM(away_goals) AS gol_tandang
FROM matches;
2-6. Tahun lahir termuda dan tertua per posisi (MIN & MAX dalam satu query).
Solusi
SELECT position,
       MIN(birth_year) AS tertua,
       MAX(birth_year) AS termuda
FROM players
GROUP BY position;
2-7. Klub kandang (home_team_id) yang rata-rata attendance-nya di atas 25.000 — WHERE vs HAVING: kenapa di sini harus HAVING?
Solusi
-- HAVING dipakai karena filternya pada HASIL agregasi (AVG),
-- bukan pada baris mentah. WHERE dijalankan SEBELUM GROUP BY.
SELECT home_team_id, ROUND(AVG(attendance)) AS rata2
FROM matches
GROUP BY home_team_id
HAVING AVG(attendance) > 25000
ORDER BY rata2 DESC;
Hari 3 · INNER JOIN dasar
🧠 Cara mikir JOIN (baca dulu sebelum ngerjain):
JOIN = "tempelkan baris tabel B ke tiap baris tabel A yang kuncinya cocok".
Contoh: players punya kolom team_id (cuma angka, misal 3). Nama klubnya ada di tabel teams. Supaya angka 3 itu berubah jadi "Yokohama Port", kita tempelkan:
FROM players p JOIN teams t ON t.team_id = p.team_id
Rumus praktis: ON selalu berisi "kolom kunci tabel A = kolom kunci tabel B". Kalau bingung mulai dari mana: (1) tulis dulu FROM tabel utama, (2) tanya "kolom apa yang kurang?", (3) JOIN tabel yang punya kolom itu lewat kunci yang sama. Alias (p, t) cuma nama pendek biar ga ngetik panjang.
3-0a. Pemanasan: jalankan SELECT * FROM players LIMIT 5; lalu SELECT * FROM teams; — perhatikan kolom team_id muncul di dua-duanya. Itulah "kunci" yang menyambungkan.
Solusi
-- Tanpa JOIN, kamu cuma lihat angka team_id:
SELECT name, team_id FROM players LIMIT 5;
-- Dengan JOIN, angka itu jadi nama klub:
SELECT p.name, p.team_id, t.name AS klub
FROM players p
JOIN teams t ON t.team_id = p.team_id
LIMIT 5;
3-0b. JOIN paling sederhana: tampilkan nama pemain + kota klubnya (cuma 2 kolom).
Solusi
SELECT p.name, t.city
FROM players p
JOIN teams t ON t.team_id = p.team_id;
3-1. Daftar pemain beserta nama klubnya.
Solusi
SELECT p.name AS pemain, t.name AS klub, p.position
FROM players p
JOIN teams t ON t.team_id = p.team_id
ORDER BY klub, pemain;
3-2. Top skor liga: 10 pemain dengan gol terbanyak.
Solusi
SELECT p.name, t.name AS klub, COUNT(*) AS gol
FROM goals g
JOIN players p ON p.player_id = g.player_id
JOIN teams t  ON t.team_id  = p.team_id
GROUP BY p.player_id
ORDER BY gol DESC
LIMIT 10;
3-3. Jumlah gol penalti per klub.
Solusi
SELECT t.name AS klub, COUNT(*) AS penalti
FROM goals g
JOIN players p ON p.player_id = g.player_id
JOIN teams t  ON t.team_id  = p.team_id
WHERE g.is_penalty = 1
GROUP BY t.team_id
ORDER BY penalti DESC;
3-4. Semua gol beserta nama pencetaknya dan menitnya, urut dari menit paling awal (JOIN 2 tabel: goals → players).
Solusi
SELECT p.name, g.minute, g.is_penalty
FROM goals g
JOIN players p ON p.player_id = g.player_id
ORDER BY g.minute
LIMIT 20;
3-5. Gol beserta TANGGAL pertandingannya (JOIN goals → matches; perhatikan kuncinya sekarang match_id, bukan player_id).
Solusi
SELECT m.match_date, g.minute, g.player_id
FROM goals g
JOIN matches m ON m.match_id = g.match_id
ORDER BY m.match_date
LIMIT 20;
3-6. Gabungkan 3-4 dan 3-5: nama pencetak gol + tanggal pertandingan + menit (JOIN 3 tabel sekaligus — goals di tengah, nyambung ke players DAN matches).
Solusi
-- goals punya player_id DAN match_id, jadi dia "jembatan" ke dua tabel:
SELECT m.match_date, p.name, g.minute
FROM goals g
JOIN players p ON p.player_id = g.player_id
JOIN matches m ON m.match_id  = g.match_id
ORDER BY m.match_date, g.minute
LIMIT 30;
3-7. Jumlah gol per klub (JOIN + GROUP BY — pola paling sering keluar di tes).
Solusi
SELECT t.name AS klub, COUNT(*) AS gol
FROM goals g
JOIN players p ON p.player_id = g.player_id
JOIN teams t  ON t.team_id  = p.team_id
GROUP BY t.team_id
ORDER BY gol DESC;
3-8. Pemain FW dari klub yang kapasitas stadionnya di atas 30.000 (JOIN + WHERE dua kondisi dari dua tabel berbeda).
Solusi
SELECT p.name, t.name AS klub, t.stadium_capacity
FROM players p
JOIN teams t ON t.team_id = p.team_id
WHERE p.position = 'FW'
  AND t.stadium_capacity > 30000
ORDER BY t.stadium_capacity DESC;
3-9. Rata-rata menit terjadinya gol per posisi pencetak (apakah FW mencetak gol lebih awal dari MF?).
Solusi
SELECT p.position, COUNT(*) AS gol,
       ROUND(AVG(g.minute), 1) AS rata2_menit
FROM goals g
JOIN players p ON p.player_id = g.player_id
GROUP BY p.position
ORDER BY gol DESC;
Hari 4 · JOIN lanjutan (alias ganda, LEFT JOIN)
4-1. Daftar pertandingan dengan NAMA tim kandang dan tandang (join teams dua kali dengan alias berbeda).
Solusi
SELECT m.match_date,
       th.name AS kandang, m.home_goals,
       ta.name AS tandang, m.away_goals
FROM matches m
JOIN teams th ON th.team_id = m.home_team_id
JOIN teams ta ON ta.team_id = m.away_team_id
ORDER BY m.match_date;
4-2. Pemain yang BELUM pernah mencetak gol (LEFT JOIN + IS NULL).
Solusi
SELECT p.name, p.position, t.name AS klub
FROM players p
JOIN teams t ON t.team_id = p.team_id
LEFT JOIN goals g ON g.player_id = p.player_id
WHERE g.goal_id IS NULL
ORDER BY klub;
4-3. Rata-rata penonton per klub saat main kandang.
Solusi
SELECT t.name AS klub,
       ROUND(AVG(m.attendance)) AS rata2,
       t.stadium_capacity
FROM matches m
JOIN teams t ON t.team_id = m.home_team_id
GROUP BY t.team_id
ORDER BY rata2 DESC;
Hari 5 · Subquery
5-1. Pemain yang lebih muda dari rata-rata umur liga.
Solusi
SELECT name, birth_year
FROM players
WHERE birth_year > (SELECT AVG(birth_year) FROM players)
ORDER BY birth_year DESC;
5-2. Pertandingan dengan total gol di atas rata-rata liga.
Solusi
SELECT match_id, match_date, home_goals + away_goals AS total
FROM matches
WHERE home_goals + away_goals >
      (SELECT AVG(home_goals + away_goals) FROM matches)
ORDER BY total DESC;
5-3. Klub yang pemainnya mencetak gol terbanyak di satu pertandingan tunggal (subquery bertingkat — soal bonus SULIT).
Solusi
SELECT t.name, sub.match_id, sub.n_gol
FROM (
  SELECT p.team_id, g.match_id, COUNT(*) AS n_gol
  FROM goals g JOIN players p ON p.player_id = g.player_id
  GROUP BY p.team_id, g.match_id
) sub
JOIN teams t ON t.team_id = sub.team_id
ORDER BY sub.n_gol DESC
LIMIT 5;
Hari 6 · CTE (WITH)
6-1. Tulis ulang soal 3-2 (top skor) memakai CTE.
Solusi
WITH skor AS (
  SELECT player_id, COUNT(*) AS gol
  FROM goals
  GROUP BY player_id
)
SELECT p.name, t.name AS klub, s.gol
FROM skor s
JOIN players p ON p.player_id = s.player_id
JOIN teams t  ON t.team_id  = p.team_id
ORDER BY s.gol DESC
LIMIT 10;
6-2. Buat klasemen sederhana: poin per klub (menang=3, seri=1) memakai CTE gabungan hasil kandang + tandang SULIT. Ini soal paling penting minggu 1 — pola klasemen sering keluar di tes.
Solusi
WITH hasil AS (
  SELECT home_team_id AS team_id,
         CASE WHEN home_goals > away_goals THEN 3
              WHEN home_goals = away_goals THEN 1 ELSE 0 END AS poin,
         home_goals AS gf, away_goals AS ga
  FROM matches
  UNION ALL
  SELECT away_team_id,
         CASE WHEN away_goals > home_goals THEN 3
              WHEN away_goals = home_goals THEN 1 ELSE 0 END,
         away_goals, home_goals
  FROM matches
)
SELECT t.name AS klub,
       COUNT(*)         AS main,
       SUM(h.poin)      AS poin,
       SUM(h.gf)        AS gol,
       SUM(h.gf - h.ga) AS selisih
FROM hasil h
JOIN teams t ON t.team_id = h.team_id
GROUP BY h.team_id
ORDER BY poin DESC, selisih DESC;
Hari 7 · Review + mini-tes (kerjakan tanpa lihat solusi, target 30 menit)
7-1. Klub dengan persentase gol dari FW tertinggi.
Solusi
SELECT t.name,
       ROUND(100.0 * SUM(CASE WHEN p.position='FW' THEN 1 ELSE 0 END)
             / COUNT(*), 1) AS pct_gol_fw
FROM goals g
JOIN players p ON p.player_id = g.player_id
JOIN teams t  ON t.team_id  = p.team_id
GROUP BY t.team_id
ORDER BY pct_gol_fw DESC;
7-2. Bulan dengan rata-rata penonton tertinggi.
Solusi
SELECT strftime('%Y-%m', match_date) AS bulan,
       ROUND(AVG(attendance)) AS rata2
FROM matches
GROUP BY bulan
ORDER BY rata2 DESC
LIMIT 1;
7-3. Pencetak gol termuda di liga (join + subquery/order).
Solusi
SELECT DISTINCT p.name, p.birth_year
FROM goals g
JOIN players p ON p.player_id = g.player_id
ORDER BY p.birth_year DESC
LIMIT 1;

MINGGU 2 — Window function & pola tes

Hari 8 · ROW_NUMBER, RANK, DENSE_RANK
8-1. Ranking top skor dengan RANK() — perhatikan bedanya dengan ROW_NUMBER saat gol sama.
Solusi
WITH skor AS (
  SELECT player_id, COUNT(*) AS gol FROM goals GROUP BY player_id
)
SELECT RANK()       OVER (ORDER BY gol DESC) AS rk,
       ROW_NUMBER() OVER (ORDER BY gol DESC) AS rn,
       p.name, s.gol
FROM skor s JOIN players p ON p.player_id = s.player_id
LIMIT 15;
8-2. Ranking pemain termuda DI DALAM tiap klub (PARTITION BY).
Solusi
SELECT t.name AS klub, p.name, p.birth_year,
       ROW_NUMBER() OVER (PARTITION BY p.team_id
                          ORDER BY p.birth_year DESC) AS rk_muda
FROM players p JOIN teams t ON t.team_id = p.team_id
ORDER BY klub, rk_muda;
Hari 9 · LAG / LEAD
9-1. Untuk tiap pertandingan kandang satu klub (pilih team_id=1), tampilkan attendance dan selisihnya dengan pertandingan kandang sebelumnya.
Solusi
SELECT match_date, attendance,
       attendance - LAG(attendance) OVER (ORDER BY match_date) AS selisih
FROM matches
WHERE home_team_id = 1
ORDER BY match_date;
9-2. Jeda hari antar-gol untuk pencetak gol terbanyak (LAG pada tanggal — gunakan julianday) SULIT.
Solusi
WITH top1 AS (
  SELECT player_id FROM goals GROUP BY player_id
  ORDER BY COUNT(*) DESC LIMIT 1
)
SELECT m.match_date, g.minute,
       CAST(julianday(m.match_date) -
            julianday(LAG(m.match_date) OVER (ORDER BY m.match_date, g.minute))
            AS INT) AS jeda_hari
FROM goals g
JOIN matches m ON m.match_id = g.match_id
WHERE g.player_id = (SELECT player_id FROM top1)
ORDER BY m.match_date;
Hari 10 · Running total & moving average
10-1. Akumulasi gol liga dari waktu ke waktu (running total per tanggal).
Solusi
WITH per_hari AS (
  SELECT m.match_date, COUNT(g.goal_id) AS gol
  FROM matches m LEFT JOIN goals g ON g.match_id = m.match_id
  GROUP BY m.match_date
)
SELECT match_date, gol,
       SUM(gol) OVER (ORDER BY match_date) AS kumulatif
FROM per_hari;
10-2. Moving average attendance 3 pertandingan terakhir (frame ROWS BETWEEN).
Solusi
SELECT match_date, attendance,
       ROUND(AVG(attendance) OVER (
         ORDER BY match_date
         ROWS BETWEEN 2 PRECEDING AND CURRENT ROW)) AS ma3
FROM matches
ORDER BY match_date;
Hari 11 · CASE, COALESCE, fungsi tanggal
11-1. Kategorikan pertandingan: 'sepi' (<15rb), 'normal' (15–30rb), 'penuh' (>30rb) lalu hitung jumlahnya.
Solusi
SELECT CASE
         WHEN attendance < 15000 THEN 'sepi'
         WHEN attendance <= 30000 THEN 'normal'
         ELSE 'penuh'
       END AS kategori,
       COUNT(*) AS n
FROM matches
GROUP BY kategori;
11-2. Daftar SEMUA pemain dengan jumlah golnya; yang belum punya gol tampil 0 (LEFT JOIN + COALESCE).
Solusi
SELECT p.name,
       COALESCE(COUNT(g.goal_id), 0) AS gol
FROM players p
LEFT JOIN goals g ON g.player_id = p.player_id
GROUP BY p.player_id
ORDER BY gol DESC;
11-3. Gol per babak: babak 1 (menit ≤ 45) vs babak 2.
Solusi
SELECT CASE WHEN minute <= 45 THEN 'Babak 1' ELSE 'Babak 2' END AS babak,
       COUNT(*) AS gol
FROM goals
GROUP BY babak;
Hari 12 · Top-N per grup (pola favorit tes!)
12-1. Top skor DI TIAP klub (1 pemain per klub).
Solusi
WITH skor AS (
  SELECT p.team_id, p.name, COUNT(*) AS gol,
         ROW_NUMBER() OVER (PARTITION BY p.team_id
                            ORDER BY COUNT(*) DESC) AS rk
  FROM goals g JOIN players p ON p.player_id = g.player_id
  GROUP BY p.player_id
)
SELECT t.name AS klub, s.name AS top_skor, s.gol
FROM skor s JOIN teams t ON t.team_id = s.team_id
WHERE s.rk = 1
ORDER BY s.gol DESC;
12-2. Dua pertandingan dengan penonton terbanyak untuk TIAP klub kandang.
Solusi
WITH ranked AS (
  SELECT m.*, ROW_NUMBER() OVER (PARTITION BY home_team_id
                                 ORDER BY attendance DESC) AS rk
  FROM matches m
)
SELECT t.name AS klub, r.match_date, r.attendance
FROM ranked r JOIN teams t ON t.team_id = r.home_team_id
WHERE r.rk <= 2
ORDER BY klub, rk;
Hari 13 · Query gaya KPI (persis kerjaan data analyst)
13-1. "Okupansi stadion": rata-rata attendance ÷ kapasitas per klub, dalam persen. KPI mana yang perlu perhatian (<60%)?
Solusi
SELECT t.name AS klub,
       ROUND(100.0 * AVG(m.attendance) / t.stadium_capacity, 1) AS okupansi_pct
FROM matches m
JOIN teams t ON t.team_id = m.home_team_id
GROUP BY t.team_id
ORDER BY okupansi_pct;
13-2. Tren bulanan: total gol dan rata-rata penonton per bulan dalam SATU tabel.
Solusi
WITH gol_bln AS (
  SELECT strftime('%Y-%m', m.match_date) AS bulan, COUNT(*) AS gol
  FROM goals g JOIN matches m ON m.match_id = g.match_id
  GROUP BY bulan
)
SELECT strftime('%Y-%m', m.match_date) AS bulan,
       ROUND(AVG(m.attendance)) AS rata2_penonton,
       gb.gol
FROM matches m
JOIN gol_bln gb ON gb.bulan = strftime('%Y-%m', m.match_date)
GROUP BY bulan
ORDER BY bulan;
13-3. Kontribusi: berapa % gol klub yang disumbang top skornya? (gabungkan pola 12-1 dengan total gol klub) SULIT
Solusi
WITH gol_pemain AS (
  SELECT p.team_id, p.name, COUNT(*) AS gol
  FROM goals g JOIN players p ON p.player_id = g.player_id
  GROUP BY p.player_id
),
klub AS (
  SELECT team_id, SUM(gol) AS total,
         MAX(gol) AS gol_top
  FROM gol_pemain GROUP BY team_id
)
SELECT t.name, k.total, k.gol_top,
       ROUND(100.0 * k.gol_top / k.total, 1) AS pct_top_skor
FROM klub k JOIN teams t ON t.team_id = k.team_id
ORDER BY pct_top_skor DESC;
Hari 14 · Simulasi tes (60 menit, tanpa solusi dulu!)
14-1. Buat klasemen akhir lengkap: main, menang, seri, kalah, gol, kemasukan, selisih, poin — urut peringkat.
Solusi
WITH hasil AS (
  SELECT home_team_id AS team_id, home_goals AS gf, away_goals AS ga FROM matches
  UNION ALL
  SELECT away_team_id, away_goals, home_goals FROM matches
)
SELECT t.name AS klub,
       COUNT(*) AS main,
       SUM(gf > ga) AS M, SUM(gf = ga) AS S, SUM(gf < ga) AS K,
       SUM(gf) AS gol, SUM(ga) AS kemasukan, SUM(gf - ga) AS selisih,
       SUM(CASE WHEN gf > ga THEN 3 WHEN gf = ga THEN 1 ELSE 0 END) AS poin
FROM hasil h JOIN teams t ON t.team_id = h.team_id
GROUP BY h.team_id
ORDER BY poin DESC, selisih DESC, gol DESC;
14-2. Pemain "super-sub": cetak gol di menit ≥ 75 paling banyak.
Solusi
SELECT p.name, COUNT(*) AS gol_akhir
FROM goals g JOIN players p ON p.player_id = g.player_id
WHERE g.minute >= 75
GROUP BY p.player_id
ORDER BY gol_akhir DESC
LIMIT 5;
14-3. Untuk tiap klub: rata-rata penonton kandang bulan ini vs bulan sebelumnya beserta % perubahan (gabungan CTE + window LAG) SULIT.
Solusi
WITH bln AS (
  SELECT home_team_id AS team_id,
         strftime('%Y-%m', match_date) AS bulan,
         AVG(attendance) AS rata2
  FROM matches GROUP BY team_id, bulan
)
SELECT t.name, b.bulan, ROUND(b.rata2) AS rata2,
       ROUND(100.0 * (b.rata2 - LAG(b.rata2) OVER
         (PARTITION BY b.team_id ORDER BY b.bulan))
         / LAG(b.rata2) OVER (PARTITION BY b.team_id ORDER BY b.bulan), 1)
         AS pct_perubahan
FROM bln b JOIN teams t ON t.team_id = b.team_id
ORDER BY t.name, b.bulan;
